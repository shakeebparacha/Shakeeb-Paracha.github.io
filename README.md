## 📖 About

This is my personal portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks required. It showcases my skills, projects, education, and certifications as a Business/Data Analyst and Power BI Specialist.

---

## ✨ Features

- 🌙 **Dark / Light Mode** — Toggle with theme memory (localStorage)
- ⌨️ **Typed Animation** — Rotating job titles in the hero section
- 📜 **Scroll Reveal** — Elements fade in as you scroll down
- 📊 **Animated Skill Bars** — Progress bars animate when in view
- 📱 **Fully Responsive** — Works on mobile, tablet & desktop
- 🔗 **Active Nav Links** — Highlights the current section
- 🃏 **Project Cards** — Hover effects with tech stack tags
- 📅 **Timeline Layout** — Vertical timeline for education history
- 📬 **Contact Form** — With submit confirmation animation
- 🔄 **Sticky Navbar** — Stays visible while scrolling
- 🎯 **Smooth Scrolling** — Seamless navigation between sections

## 🚀 Getting Started

### Option 1 — Run Locally (Simplest)

```bash
# 1. Clone the repository
git clone https://github.com/Shakeeb-Paracha/Shakeeb-Paracha.github.io.git

# 2. Navigate to the project folder
cd Shakeeb-Paracha.github.io

# 3. Open index.html in your browser
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

### Option 2 — VS Code Live Server

1. Open the project folder in **VS Code**
2. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
3. Right-click `index.html` → **"Open with Live Server"**
4. Visit `http://127.0.0.1:5500` in your browser ✅

### Option 3 — Online Editor (No Install)

1. Go to [CodePen](https://codepen.io/pen/)
2. Paste contents of `index.html`, `style.css`, and `script.js` into respective panels
3. Preview instantly ✅

## 🌍 Deployment

### GitHub Pages (Free Hosting)

1. Create a repository named **`Shakeeb-Paracha.github.io`**
2. Upload all project files (`index.html`, `style.css`, `script.js`)
3. Go to **Settings → Pages**
4. Set source to `main` branch → `/ (root)` → Click **Save**
5. Your site is live at `https://shakeebparacha.github.io/Shakeeb-Paracha.github.io/` 🎉
